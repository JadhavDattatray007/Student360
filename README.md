# Student360 - AI Powered Student Holistic Development System

## 📌 Problem Statement

Traditional student evaluation focuses mainly on academic performance. However, true student development includes academics, sports, leadership, innovation, communication skills, certifications, and overall personality growth.

Student360 is designed to provide a 360-degree view of student development aligned with NEP 2020 objectives.

---

## 🎯 Objectives

* Track academic performance
* Track sports participation
* Track certifications
* Track leadership activities
* Track innovation projects
* Track communication skills
* Generate holistic student profiles
* Support AI-based development recommendations
